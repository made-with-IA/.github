---

## 🚀 Projects

### 📚 [Cond-Library-h26](https://github.com/made-with-IA/Cond-Library-h26) — Condominium Library

A full-stack community library management system built for condominiums.

**Highlights:**
- Public catalog with book search, reservations, and loan tracking
- Admin panel with real-time metrics, overdue alerts, and WhatsApp notifications
- AI-powered book lookup via **Gemini AI** (auto-fill book data by ISBN or title)
- Reservation queue system with automatic expiry and next-in-line promotion
- Built with **React 19**, **Express 5**, **PostgreSQL**, **Drizzle ORM**, and **TypeScript**

---

### 🎲 [sorteio-meet](https://github.com/made-with-IA/sorteio-meet) — Meeting Raffle

A lightweight, client-side raffle tool perfect for team meetings and events.

**Highlights:**
- Create private raffles with a unique shareable link
- QR Code generation for quick in-person participation
- Admin dashboard: open/close registrations, set winner count, draw instantly
- No backend needed — all data persisted via `localStorage`
- Built with **React**, **Vite**, **Tailwind CSS**, and **TypeScript**
