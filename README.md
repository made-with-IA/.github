## About made-with-IA

**made-with-IA** is an open organization building real-world applications with AI as a collaborative development partner. We explore how artificial intelligence can accelerate the full software lifecycle — from ideation and design to coding and deployment.

### Projects

| Project | Description |
|---------|-------------|
| [Cond-Library-h26](https://github.com/made-with-IA/Cond-Library-h26) | Full-stack condominium library management system with AI-powered book search |
| [sorteio-meet](https://github.com/made-with-IA/sorteio-meet) | Client-side raffle tool for team meetings and events with QR Code support |

### Ideas in Progress

We are always thinking about new tools to build. Current concepts include an AI voting app, a smart task board, an expense splitter, and a meeting scheduler — all designed to solve everyday team and community problems with a touch of AI.

Feel free to open an issue to suggest a project idea or start a discussion!
